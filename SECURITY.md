# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.x     | Yes     |
| <1.0.0     | No      |

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not open a public issue**.

Instead, report it privately by contacting the maintainer:

* Email: [gourabanandadatta@zohomail.in](mailto:gourabanandadatta@zohomail.in)

Please include:

* A description of the issue
* Steps to reproduce
* Potential impact

## What to Expect

* You will receive an acknowledgment within a few days
* We will investigate and work on a fix
* Once resolved, the fix will be released and documented

Thank you for helping keep this project secure
